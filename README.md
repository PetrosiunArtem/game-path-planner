# Progress Path Planner (MVP)

Это одностраничное приложение (SPA) для планирования стратегии прохождения игры Cuphead.

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```

## Тестирование
Для запуска тестов (заглушка):
```bash
npm test
```
